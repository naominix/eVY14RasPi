eVY14RasPi
==========

eVY1 Scratch on Raspberry Pi for Pokemiku NSX-39 Installer

## Installation

Install eVY1 Scratch.

```
# curl https://naominix.github.io/pokemikuinstall.sh | sh
```

Overwrite so.MIDIPlugin file.
